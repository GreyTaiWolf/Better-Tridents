# 附灵三叉戟使用规范

## 基础状态

- `FLYING`：刚投出的三叉戟飞行状态。
- `ORBIT`：附灵三叉戟悬浮在玩家周围，是主要待命状态。
- `ATTACKING`：玩家攻击可攻击生物后，三叉戟自动追击目标。
- `CHARGING`：空主手长按左键后进入蓄力姿态。
- `CHARGED_SHOT`：蓄力成功后松开左键，三叉戟高速发射。
- `WAITING_FOR_OWNER`：特殊情况下落地等待玩家靠近回收。

## 操作规则

- 空主手长按左键：开始蓄力。
- 长按 3 秒：进入蓄力姿态，但还没有蓄力等级。
- 继续蓄力到 1/2/3 级：分别获得约 100/200/300 格最大飞行距离。
- 蓄力达到 1 级后：准星瞄准的可攻击生物会短暂发光，表示当前锁定目标。
- 蓄力时移动准星：高亮会切换到新的可攻击生物。
- 蓄力成功后松开左键：按当前等级发射。
- 有高亮目标时松开左键：优先朝高亮目标发射，并在飞行中轻微修正方向。
- 没有高亮目标时松开左键：按准星方向直线发射。
- 未达到 1 级时松开左键：取消蓄力并回到悬浮。
- 空主手右键准星指向悬浮三叉戟本体：收回三叉戟。
- 右键空气、方块、其他生物：不收回悬浮三叉戟。

## 自动攻击规则

- 玩家攻击可攻击生物后，悬浮三叉戟会锁定该目标。
- 可攻击目标为非玩家 `LivingEntity`，包括怪物、动物和村民。
- 不主动攻击玩家，也不会主动攻击三叉戟主人。
- 命中后不会原地硬折返，而是继续惯性飞出，再逐渐转向回击。
- 目标死亡、失效或超出范围后，三叉戟回到悬浮状态。
- 玩家再攻击新的可攻击生物后，三叉戟应重新锁定新目标。

## 回收规则

- 只有 `ORBIT` 悬浮状态可以被空主手右键收回。
- 蓄力中、蓄力发射中、自动攻击中不会被右键收回。
- 创造模式且三叉戟为创造拾取规则时，收回不会额外返还物品。
- 生存模式收回时优先放入背包，背包满时掉落到玩家附近。

## 蓄力等级

| 等级 | 达成时间 | 最大距离 |
| --- | --- | --- |
| 1 级 | 长按约 5 秒 | 100 格 |
| 2 级 | 长按约 8 秒 | 200 格 |
| 3 级 | 长按约 11 秒 | 300 格 |

说明：前 3 秒用于进入蓄力姿态，随后继续蓄力才获得等级。

## 测试清单

- 投出附灵三叉戟后确认它进入悬浮状态。
- 悬浮移动应平滑，重新进入世界后不应明显卡顿。
- 空手右键空气不收回。
- 空手右键准星指向悬浮三叉戟本体可以收回。
- 空手长按左键可以蓄力，松开后按等级发射。
- 蓄力 1 级后准星瞄准动物、怪物或村民，应出现发光锁定。
- 准星切换到另一个可攻击生物后，高亮目标应同步切换。
- 有高亮目标时松开发射，应优先攻击高亮目标。
- 攻击第一只生物并让三叉戟击杀后，再攻击第二只生物，三叉戟应继续响应。
